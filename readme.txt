Rewrite the root url to CDN URL and cache them to CDN server.
Cache9 코리아의 대한민국 전용 CDN 서비스용 도메인 변경 플러그인.

== Description ==

[Cache9 Korea](http://kr.cache9.com/) CDN provied by Korean local servers for Korean users.
Rewrite the host of css, js, image, multi-media files for Cache9 CDN and any others(Origin pull / Mirror).
This plugin is for[Cache9 Korea](http://kr.cache9.com/ "Korean only"), and the English and Chinses site will comming soon.

= 대한민국 최초 국내 전용 서버를 이용한 워드프레스용 CDN =

[Cache9(캐시9) 코리아](http://kr.cache9.com/) CDN은 대한민국 전용 서버를 이용한 워드프레스용 CDN 서비스를 제공합니다.
대한민국 최초로 제공되는 대한민국내 전용 서버를 이용한 워드프레스용 CDN으로 더 이상 느리고 비싼 해외 서비스를 이용하지 않으셔도 됩니다.

css, js, 이미지, 멀티미디어 파일 주소 변경 기능으로 Cache9(캐시9) CDN이나 원본 서버 풀(캐싱), 미러 형식의 CDN 서비스를 사용할 수 있습니다.

이 플러그인은 [Cache9(캐시9) 코리아](http://kr.cache9.com/ "한국어 사이트")용으로 제작되었으며, 다국어 사이트(영어, 중국어)도 곧 오픈될 예정입니다.

이 플러그인은 W3 Total Cache, WP Super Cache 등의 플러그인과 함께 사용할 수 있습니다.


= Cache9(캐시9) CDN은 FTP 업로드가 필요 없습니다 =

이 플러그인을 설치하고 활성화 시키는 동시에 CDN 서비스가 가능합니다.


= 플러그인 설치와 함께 7일 무료 체험 기간이 제공됩니다 =

플러그인 설치와 함께 7일 무료 체험 기간이 제공되며, [Cache9(캐시9) 코리아](http://kr.cache9.com/)에서 포인트를 충전하여 서비스를 계속 사용하실 수 있습니다.

자세한 내용은 설치 방법(Installation)을 참고해주세요.


= 파일 지원 =

Cache9(캐시9) CDN은 다음과 같은 파일들을 지원합니다.

* CSS 파일 : .css
* 자바스크립트 파일 : .js
* 이미지 파일 : .gif, .jpg, .jpeg, .png, .bmp, .tiff, .jpc, .jp2, .jpf, .jb2, .aiff, .wbmp, .xbm
* 폰트 파일 : .eot, .ttf, .otf, .woff, .svg
* 멀티미디어 파일 : .mp3, .mp4


= Cache9(캐시9) 글로벌 =

Cache9(캐시9) 글로벌과 Cache9(캐시9) 차이나와 같은 다국어(영어, 중국어) 서비스 사이트도 곧 출시될 예정입니다.


== Installation ==

Cache9(캐시9) CDN 플러그인은 설치와 이용이 간단합니다.

1. `cache9-cdn` 폴더를 `/wp-content/plugins/` 디렉토리에 업로드 하거나, 워드프레스 관리자의 '플러그인 추가하기' 메뉴에서 검색, 설치할 수 있습니다.
1. 워드프레스 관리자의 '설정 | Cache9 CDN' 메뉴에서 신규 계정(7일간 무료 체험 포함)을 발급 받으시거나, 기존에 사용하시던 Cache9 CDN 계정을 사용할 수 있습니다.
1. 워드프레스 관리자의 '설정 | Cache9 CDN' 메뉴에서 CDN에 적용할 파일 종류, 워드프레스 경로 등을 설정하고, 주소 변경 활성화를 시키면 CDN 서비스를 이용할 수 있습니다.
1. CDN 플랜, 지역 등의 옵션 설정이나 서비스 연장이 필요할 경우 [Cache9(캐시9) 코리아](http://kr.cache9.com/ "The Global and Asia optimized CDN such as Korea and China.")에서 [회원 등록](http://kr.cache9.com/register/ "Cache9(캐시9) 코리아(한국어)") 후 가능합니다..


== Frequently Asked Questions ==

= 플러그인 사용은 무료인가요? =

네, Cache9 CDN 플러그인 사용은 무료이지만, CDN 서비스를 이용하시려면 CDN 사용 비용을 결제해야 합니다.
단, 이 플러그인은 7일 무료 체험 기간이 함께 제동되므로 7일간 CDN 서비스를 무료로 이용해보실 수 있습니다.
Cache9 CDN 비용은 사용한 만큼 과금되도록 책정되어 타사 대비 저렴하게 이용 가능합니다.
자세한 내용은 [CDN 비용 계산기](http://kr.cache9.com/cdn-price-korea-calculator/. "한국어")를 참고하세요.

= 플러그인을 설치하면 CDN 서비스를 바로 사용할 수 있나요? =

네, 이 플러그인은 7일 무료 체험 기간이 함께 제공되며, 원하실 경우 포인트 충전으로 서비스 연장이 가능합니다.
단, 최초 설정시 발급된 CDN 주소의 DNS 서버 적용 시간이 2~3분 이상 소요될 수 있습니다.

= Cache9(캐시9) 코리아는 국내 전용 CDN만 제공하나요? =

Cache9(캐시9) 코리아는 글로벌 서비스를 제공하며, Cache9(캐시9) 코리아 서비스 사이트 언어만 한글로 되어있으며, 대한민국 전용, 중국(대륙)전용, 글로벌 CDN을 선택하여 이용할 수 있습니다.
오픈 예정인 Cache9(캐시9) 글로벌과 Cahce9(캐시9) 차이나를 이용하셔도 동일한 서비스를 이용할 수 있습니다.
